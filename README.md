# UIKitStuff

A collection of UIKit learning projects demonstrating various iOS development concepts and patterns.

## Projects

### CompositionalLayout
A project showcasing UICollectionViewCompositionalLayout for building complex, flexible collection view layouts.

### DiffableDatasource
A project demonstrating the use of UICollectionViewDiffableDataSource for efficient data management and UI updates in collection views.

## Requirements

- iOS 26.4 or later
- Xcode 16.0 or later
- Swift 5.0 or later

## Getting Started

1. Clone the repository
2. Open `UIKitStuff.xcworkspace` in Xcode
3. Select the desired project/target
4. Build and run on a simulator or device
